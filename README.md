# nlp-stopwords-punctuation

## 📌 Overview
This project demonstrates basic NLP preprocessing using Python and NLTK.

It shows how to:
- Load stopwords using NLTK
- Display English stopwords list
- Display punctuation symbols

---

## 🧹 Stopwords
Stopwords are commonly used words like *is, the, and, in* that do not add much meaning to a sentence.  

---

## 🔤 Punctuation
Punctuation refers to special characters like . , ! ? : ; used in writing to structure sentences.  

---

## 📦 Requirements
- Python 3.x
- NLTK library
